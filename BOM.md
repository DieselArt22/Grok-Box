# Grok-Box Bill of Materials (BOM)

## Core Server (Main Box)

### GPU
- RTX 3090 24GB (used) - ~$800-1050

### CPU
- AMD Ryzen 7 9700X or similar efficient AM5 CPU

### Motherboard
- Mini-ITX B650 (e.g. ASUS ROG Strix B650I)

### RAM
- 64GB or 128GB DDR5-6000 (2x32GB or 2x64GB)

### Case
- Fractal Design Terra or NR200P (SFF, fits 3090)

### PSU
- 850W+ SFX Gold/Platinum (e.g. Corsair SF)

### Storage
- 1TB+ NVMe SSD

### Cooling
- Good air cooler or optional AIO for GPU

## Satellites (Per Room)
- ESP32 or HA Voice Preview devices
- 3D printable enclosures (PA6-CF)

## Software (All Open Source/GPL)
- Home Assistant
- Ollama + local models
- NVIDIA PersonaPlex
- faster-whisper, Piper TTS
